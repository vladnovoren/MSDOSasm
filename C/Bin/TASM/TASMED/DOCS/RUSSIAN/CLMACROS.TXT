
     Зачем нужны макросы командной строки?
     ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
         Макросы  командной  строки  (в  дальнейшем  просто  макросы)  были
     добавлены  в  TASMED  чтобы  сделать  возможным интерфейс с различными
     ассемблерами, компоновщиками, отладчиками и различными утилитами.

     Макросы определенные при ассемблировании
     ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
      $ASM_NAME - имя файла для ассемблирования
      $OBJ_NAME - имя целевого объектного файла
      $LST_NAME - имя листинга или пусто, если листинг отключен
      $LST - ключ для включения генерации листинга
      $COMMA_LST - запятая, если листинг включен (для TASM-а)
      $DEBUG - ключ для включения генерации отладочной информации
      $INCLUDES - путь к включаемым файлам из диалога "Project options"
      $CONDS - условные определения из диалога "Project options"

         Стандартная строка для Turbo Assembler-а:
     $ASM_NAME, $OBJ_NAME $COMMA_LST$LST_NAME $LST $DEBUG $INCLUDES
     $CONDS

         Стандартная строка для Microsoft Assembler-а:
     /nologo $ASM_NAME /Fo$OBJ_NAME $LST$LST_NAME $DEBUG /c $INCLUDES
     $CONDS

     Макросы определенные при компоновке
     ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
      $OBJ_NAMES - список объектных файлов через пробел
      $LIB_NAMES - список библиотек через пробел
      $PRIMARY - имя запускаемого модуля
      $MAP - ключ для включения генерации .MAP файлов
      $DEBUG - ключ для включения генерации отладочной информации
      $TARGET - зависит от типа запускаемого модуля (.COM или .EXE)

         Стандартная строка для Turbo Linker-а:
     $OBJ_NAMES, $PRIMARY, , $LIB_NAMES $TARGET $MAP $DEBUG /3

         Стандартная строка для Microsoft Linker-а:
     /nologo $OBJ_NAMES /Fe$PRIMARY /link /nologo $LIB_NAMES $TARGET $MAP
     $DEBUG

     Макросы определенные для отладчика
     ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
      $PRIMARY - имя запускаемого модуля
      $PARAMS - параметры

         Стандартная строка для Turbo Debugger-а:
     TD.EXE $PRIMARY $PARAMS

     Макросы определенные для внешних утилит
     ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
      $CUR_NAME - полное имя файла в текущем окне редактора
      $CUR_TOKEN - слово на котором находится курсор
      $COL - номер столбца на котором находится курсор
      $LINE - номер строки на которой находится курсор
      $PRIMARY - имя запускаемого модуля
      $PARAMS - параметры

     Макрос определенный для фильтра печати
     ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
      $CUR_NAME - полное имя файла в текущем окне редактора

         Стандартная строка:
     $TASMED_PATH\FILTERS\PRINT.EXE $CUR_NAME /EPSON

     Прочие макросы
     ~~~~~~~~~~~~~~
      $TASMED_PATH - путь где располагается TASMED.COM (без последнего \)
